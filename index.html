🔗 Connecter Airtable

Base de données partagée pour toutes vos équipes en temps réel.

  1. Créez un compte sur airtable.com
  2. Créez une base "DIWALGUI" → table "Dépôts"
  3. Colonnes : Agent, Téléphone, Banque, Montant, Date, Référence, Statut, Spécialiste, Commentaire, Validé par, Motif Rejet
  4. Account → Developer Hub → Personal Access Token
  5. Copiez l'ID de base (commence par app…)
